## How To Run
  ### Open .env and your api-key setting
  ```
  REACT_APP_APIKEY={YOUR_NYTIMES_APIKEY}
  ```

  ### CRA(typescript)
  ```
  $ cd client
  $ npm install
  $ npm run start # running on http://localhost:3000
  ```
